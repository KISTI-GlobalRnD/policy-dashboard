## 현재 리스크

1. `12개 정책 문헌`이라는 이름과 실제 ZIP 내부 파일 구성이 일치하지 않는다.
2. 정책 구성안 PDF는 텍스트 추출이 쉽지 않은 이미지형 페이지가 많아 OCR 또는 비전 기반 추출이 필요하다.
3. 기술분야 개요 ZIP은 11개 파일만 있어 기술 기준표의 14개 대분류를 모두 설명하지 못한다.
4. 해외 정책 자료는 원문만 있고 국내 기준 체계로 정리된 비교표가 없다.
5. 현재 문서들은 대시보드 입력 데이터가 아니라 원문/초안 단계의 자료다.

## 우선 결정해야 할 사항

1. `260305 범부처 기술관리체계 정비 및 협업 강화 방향(안)`을 분석 범위에 포함할지 결정
2. 민주당 공약집 PDF를 정책 분석 범위에 포함할지 결정
3. `15개 전략`을 공식 기준 축으로 사용할지 확정
4. 기술분야 기준의 authoritative source를 엑셀 파일로 고정할지 확정
5. 표와 그림을 어떤 수준까지 구조화할지 결정
