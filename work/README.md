# work

단계별 작업 산출물을 저장하는 영역이다.

## 폴더 사용 규칙

- `01_scope-and-ia/`
  - 범위 정의, 질문 목록, 화면 정보설계, 의사결정 로그
- `02_structured-extraction/`
  - 텍스트, 표, 그림, 레이아웃, 추출 매니페스트
- `03_processing/`
  - 정규화 데이터, 매핑 테이블, 대시보드용 마트
- `04_ontology/`
  - 스키마, 어휘집, 인스턴스 또는 그래프 산출물
- `05_dashboard/`
  - 화면 요구사항, 와이어프레임, 데이터 계약
- `06_integration/`
  - 최종 내보내기, 연결 결과, 통합 검증 기록

## 저장 원칙

- 원본 재배치보다 파생 산출물 저장에 집중한다.
- 파일명에는 날짜와 버전을 포함하는 편이 좋다.
- 근거 추적이 필요한 파일은 원문 문서명과 페이지 정보를 함께 남긴다.
