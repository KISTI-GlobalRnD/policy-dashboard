# 핵심 질문 매트릭스 초안

## 목적

대시보드가 실제로 답해야 할 질문을 먼저 고정한다.

## 질문 후보

| ID | 질문 | 필요한 데이터 | 주요 보기 |
| --- | --- | --- | --- |
| Q1 | 어떤 정책이 어떤 전략 축에 연결되는가 | 정책항목, 전략 매핑, 근거 블록 | 전략별 보기, 정책 상세 |
| Q2 | 각 정책은 `기술 / 인프라 및 제도 / 인재` 중 어디에 무게가 실려 있는가 | 정책항목, 자원유형 분류 | 정책별 보기, 비교 차트 |
| Q3 | 어떤 기술분야에 정책 투자가 집중되는가 | 기술 대분류/중분류 매핑, 문서별 항목 수 | 기술분야별 보기 |
| Q4 | 특정 전략이나 기술분야의 근거는 원문 어디에 있는가 | 문서 경로, 페이지, 텍스트 블록, 표, 그림 | 원문 근거 보기 |
| Q5 | 국내 정책 묶음과 해외 정책은 어떤 지점에서 대응되는가 | 국내 전략, 해외 정책 요약, 비교 매핑 | 해외 비교 |
| Q6 | 특정 부처가 주도하는 정책 축은 무엇인가 | 발행 기관, 정책항목, 전략 매핑 | 기관별 보기 |

## 초기 필수 필터

- 문서명
- 발행 기관
- 발행 시점
- 전략
- 기술 대분류
- 기술 중분류
- 자원유형
- 국내/해외 구분

## 이번 단계에서 정할 것

- 어떤 질문을 1차 출시 범위에 넣을지
- 어떤 질문은 2차 고도화로 미룰지
- 질문별 최소 데이터셋이 무엇인지
