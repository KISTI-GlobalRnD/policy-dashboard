# 골든 샘플 목록 v0.1

## 목적

전체 문서 전에 형식별 추출 난이도를 검증할 대표 샘플을 정한다.

## P0 샘플

### GS-001

- 문서: `DOC-REF-001`
- 제목: 정책-항목 구성(안)
- 형식: PDF
- 선정 이유: 이미지형 페이지, 표 중심, 텍스트 추출 난이도 높음
- 검증 포인트:
  - 표 구조 인식
  - 한글 OCR 품질
  - 페이지별 제목 추출

### GS-002

- 문서: `DOC-POL-006`
- 제목: 초혁신경제 15대 프로젝트 추진계획
- 형식: PDF
- 선정 이유: 정책 원문 PDF 계열의 기준 샘플
- 검증 포인트:
  - 본문 블록 추출
  - 표/도표 존재 여부 확인
  - 페이지 기반 근거 연결

### GS-003

- 문서: `DOC-POL-004`
- 제목: 과학기술xAI 국가전략
- 형식: HWPX
- 선정 이유: 정책 핵심 문헌이며 HWPX 처리 경로 검증 필요
- 검증 포인트:
  - 문단/표/이미지 분리 추출
  - 제목 및 항목 구조 보존
  - 텍스트와 근거 블록 키 연결

### GS-004

- 문서: `DOC-TAX-001`
- 제목: 중장기투자전략 세부기술분야별 중분류
- 형식: XLSX
- 선정 이유: 기술 기준표의 authoritative source
- 검증 포인트:
  - 시트 구조 추출
  - 머리글과 병합 구조 보존
  - 대분류/중분류 키 생성

## 파일럿 성공 기준

- P0 샘플 4건에서 문서 매니페스트 생성 성공
- 각 샘플에서 텍스트 또는 셀 구조가 재사용 가능한 수준으로 저장됨
- 실패 유형이 명시적으로 분류됨
- QA 체크리스트의 추출 항목을 통과함
